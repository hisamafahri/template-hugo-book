# Hisam's Hugo Book Template

This is a *bare-minimum* template to create a [Hugo Book](https://github.com/alex-shpak/hugo-book/) docs site.

## Get Started

To get started with creating a site, simply click "[use this template](https://github.com/new?template_name=template-hugo-book&template_owner=hisamafahri)" to create a GitHub repository.
